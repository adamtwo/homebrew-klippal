# Homebrew Tap for KlipPal

This is the official Homebrew tap for [KlipPal](https://github.com/adamtwo/klippal), a native macOS clipboard manager.

## Installation

```bash
brew tap adamtwo/klippal
brew install --cask klippal
```

Or in a single command:

```bash
brew install --cask adamtwo/klippal/klippal
```

## Updating

```bash
brew upgrade --cask klippal
```

## Uninstalling

```bash
brew uninstall --cask klippal
brew untap adamtwo/klippal  # optional
```

## About KlipPal

KlipPal is a native macOS clipboard manager with:
- Local-only storage (no cloud sync)
- Fast search with global hotkey access (Cmd+Shift+V)
- Support for text, URLs, images, and files
- Code signed and notarized by Apple

For more information, visit the [KlipPal repository](https://github.com/adamtwo/klippal).
